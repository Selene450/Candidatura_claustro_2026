# 🗳️ Campaña Electoral Claustro UVigo 2026 - Ana Álvarez López

Benvido/a ao repositorio oficial da miña candidatura para o **Claustro da Universidade de Vigo**. Este proxecto aloxa a páxina web onde presento os meus eixos de acción para mellorar a nosa universidade.

##  Sobre o Proxecto
O obxectivo desta web é ofrecer un espazo transparente e accesible para que todo o alumnado coñeza as propostas da miña candidatura. Baixo o lema **"A túa voz é o meu voto"**, busco unha representación activa e comprometida.

## 📌 Eixos de Acción
A miña proposta céntrase en tres piares fundamentais:
* **Políticas anti-acoso:** Medidas de protección real e espazos seguros.
* **Dereito a paro académico:** Facilitar o exercicio real do dereito á folga estudantil.
* **Inclusividade:** Unha universidade de acollida para todos e todas.

## 🛠️ Tecnoloxías utilizadas
Para o desenvolvemento desta páxina utilicei:
* **HTML5/CSS3:** Estrutura e deseño limpo e responsive.
* **Google Fonts (Inter):** Para unha lexibilidade moderna e profesional.
* **GitHub Pages:** Aloxamento gratuíto e eficiente.
